# <center> MsBuilder CLI </center>

## Build Hapi.js micro services template
- folders and startup
- auto-swagger
- eslint
- dockerfile
- tests script (jest)

## How to use?
```bash
    npm i .
    npm link # linux only
    ms-builder
```

### OPTIONS
    --template
        micro service template (hapi or express)
    --name 
        micro service name
